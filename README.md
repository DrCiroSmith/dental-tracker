# Dental Shadowing Tracker

![Test and Deploy](https://github.com/DrCiroSmith/dental-tracker/actions/workflows/test-and-deploy.yml/badge.svg)

A comprehensive web application for tracking dental shadowing, volunteering, and extracurricular hours for dental school applications (specifically tailored for CAAPID requirements).

## Features

-   **Track Hours:** Log shadowing, dental volunteering, and non-dental volunteering hours.
-   **Interactive Map:** Visualize clinic locations.
-   **Progress Dashboard:** Track progress towards 350-hour goal.
-   **Export Data:** Export logs to CSV/ZIP for applications.
-   **Offline Capable:** PWA support for mobile usage.
